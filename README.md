# Library Management System

A console-based Library Management System built using Java.

## 📌 Features
- Add Book
- Remove Book
- Search Book
- Display All Books
- Menu-driven CLI

## 🛠 Technologies Used
- Java
- OOP (Encapsulation)
- ArrayList
- Scanner
- Switch-case

## 🚀 How to Run
1. Clone the repository
2. Open in IntelliJ IDEA
3. Run Main.java
